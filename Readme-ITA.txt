MiniFTP è un servizio di trasferimento di file: il cliente invia al server 
il nome di un file (in formato testo) preso dalla linea di comando. 
Il server risponde mandando al cliente il contenuto del file, riga per riga.
Il cliente salva il contenuto in un file locale con lo stesso nome. 
Sono gestite situazioni di errore (file not found, etc). 
Il server è in grado di gestire più client alla volta.
